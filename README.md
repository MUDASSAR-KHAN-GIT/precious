# 🌸 PRECIOUS-MD 🍂 – Ultimate WhatsApp Bot

Multi-session bot with 200+ commands: media download, AI, games, group management, automation.

## Setup
1. Clone repo, run `npm install`
2. Set `MONGODB_URI` in `.env`
3. Place your audio file at `audio/dev.mp3`
4. Run `npm start`

## Commands
Send `.menu` in WhatsApp to see all categories. Use `.menu <category>` for detailed list.

## Deployment
Heroku: add `MONGODB_URI` config var, deploy branch.
